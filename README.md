# Uzair M. Umair Portfolio

Premium 3D personal portfolio for Uzair M. Umair, Graphic Designer and Full Stack Web Developer.

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- React Three Fiber
- Three.js

## Local Setup

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Deploy

This project is ready for Vercel. The build output directory is `dist`.
